# Heart Attack Analysis & Prediction

## Overview

This project analyzes a heart attack dataset and explores the relationship between different health indicators and the likelihood of heart disease. The project combines exploratory data analysis (EDA), data preprocessing, and basic predictive analysis to identify potential risk factors associated with heart attacks.

## Dataset

The dataset contains medical information collected from 303 patients, including demographic and health-related attributes such as:

* Age
* Sex
* Chest Pain Type (cp)
* Resting Blood Pressure
* Cholesterol Level
* Fasting Blood Sugar
* Resting ECG Results
* Maximum Heart Rate Achieved
* Exercise Induced Angina
* Number of Major Vessels (ca)

Target Variable:

* 0 = Lower chance of heart attack
* 1 = Higher chance of heart attack

Dataset Source:
[https://www.kaggle.com/datasets/rashikrahmanpritom/heart-attack-analysis-prediction-dataset](https://www.kaggle.com/datasets/sonialikhan/heart-attack-analysis-and-prediction-dataset)


## Project Objectives

* Analyze factors related to heart disease.
* Explore correlations between medical features and heart attack risk.
* Handle missing values using Mean Imputation.
* Visualize patterns and trends in the dataset.
* Build a foundation for heart attack prediction using data analysis techniques.

## Data Preprocessing

The dataset was cleaned and prepared before analysis.

Techniques used:

* Missing value handling using Mean Imputation.
* Data exploration and statistical analysis.
* Correlation analysis between features and target values.

## Key Findings

* Chest pain type showed a positive relationship with heart attack occurrence.
* Exercise-induced angina appeared less frequently among heart attack cases in the analyzed dataset.
* Several health indicators demonstrated varying levels of correlation with the target variable.
* Visual analysis helped identify trends and patterns among patient groups.
